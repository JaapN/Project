World Maps
When scrolling over a country on the map, information about the country name and data will pop up.
The borders of a country will get brightly coloured as well, which will make it clear which country you are looking at. There is a clear contrast between the used fill colours and highlighted border colour.

Grouped Barcharts
Click on a country to draw and display the respective educational field distribution barchart below the map and select menu. Do note however that data about educational field distribution is not available for all countries, even some of those for which there is data to colour the respective country with. When attempting to select a country for which there is no education field data, no barchart will be displayed. The educational field data represents the distribution of the total tertiary education (levels 5 to 8) of that country. The data for the Netherlands and Poland has been extrapolated from the known distribution of Bachelor's (level 6) and Master's (level 7) level graduates.

Plots
Select a combination of variables to have them plotted against each other.
